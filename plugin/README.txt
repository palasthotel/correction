== Installation ==

1. Upload `corrections.zip` to the `/wp-content/plugins/` directory
1. Extract the Plugin to a `corrections` Folder
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Use the corrections section in the gutenberg editor
